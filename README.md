# Smart Healthcare Assistant

This project predicts diseases like diabetes using Machine Learning.

## Features
- Disease prediction using ML models
- Explainable AI (SHAP, LIME)
- Data preprocessing & feature engineering

## Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Flask

## Description
This system analyzes patient data such as BMI, age, glucose levels to predict disease risk.
